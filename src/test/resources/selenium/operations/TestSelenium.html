Verify WebSite Login

Once the customer is registered to the Bank, the Bank creates the user online account. Therefore, the user should be able to perform login to the website and check if account is activated.

The system therefore attempts to login with the valid user credentials.

Example

The Customer's Bank User Id admin will be entered in User Name field And the secret password admin123 will be typed into password field.

With the provided credentials, user submits login and the result will be Success

And the welcome page will show as Welcome Admin
